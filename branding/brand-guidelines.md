# Nexus Brand Guidelines

## Name

**Nexus** — A unified media automation platform.

## Creator

spontaneocus

## Visual Identity

### Style
- Modern, sleek, minimalist, technical
- Premium control plane aesthetic
- Clean lines, geometric precision

### Logo Concept
- Connected-node abstract "N" inside a soft geometric mark
- Represents the interconnection of multiple services into one unified hub
- Minimal line weight, no unnecessary detail

### Logo Files

| File                    | Purpose                              |
|-------------------------|--------------------------------------|
| `nexus-logo.svg`        | Primary full logo (horizontal)       |
| `nexus-mark.svg`        | Icon mark only (square, for avatars) |
| `nexus-logo-dark.svg`   | Logo optimised for dark backgrounds  |
| `nexus-logo-light.svg`  | Logo optimised for light backgrounds |
| `favicon.svg`           | Browser tab favicon                  |
| `app-icon-512.png`      | PWA / app icon (512×512)             |

## Typography

- **Headings**: Inter (600, 700 weight)
- **Body**: Inter (400, 500 weight)
- **Monospace**: JetBrains Mono (for logs, code, API keys)

## Spacing and Layout

- Follow Tabler's spacing scale
- 8px base grid
- Cards with subtle borders and soft shadows
- Rounded corners (6px default, 8px for cards, 12px for modals)

## Iconography

- Use Tabler Icons as the primary icon set
- Consistent 20px or 24px icon size in navigation
- Outlined style for nav, filled style for status indicators
